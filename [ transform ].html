/* IE 9 */ -webkit-transform:translate(50px, 100px); transform:translate(50px, 100px); translate(), translateX(), translateY() rotate() rotate() method rotates an element clockwise or counter-clockwise according to a given degree. -ms-transform:rotate(20deg); /* IE 9 */ -webkit-transform:rotate(20deg); transform:rotate(20deg); scale() scale() method increases or decreases the size of an element (according to the parameters given for the width and height). -ms-transform: scale(0.5, 0.5); /* IE 9 */ -webkit-transform: scale(0.5, 0.5); /* Safari */ transform: scale(0.5, 0.5); scale(), scaleX(), scaleY() skew() skew() method skews an element along the X and Y-axis by the given angles. -ms-transform:skew(20deg, 20deg); /* IE 9 */ -webkit-transform:skew(20deg, 20deg); transform:skew(20deg, 20deg); if the second parameter is not specified, it has a zero value. skewX() method skews an element along the X-axis by the given angle. skewY() method skews an element along the Y-axis by the given angle. matrix() matrix() method combines all the 2D transform methods into one. matrix() method take six parameters, containing mathematic functions, which allows you to rotate, scale, translate, and skew elements. the parameters are as follow : matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()) -- 6 values 3D transforms : rotateZ() rotateZ() method rotates an element around its z-axis at a given degree trnsform-origin : transform-origin property allows you to change the poisition of transformed elements. transform-style : let the transformed child elements preserve the 3D transformations. transform-style property specifies how nested elements are rendered in 3D space. This property must be used together with the transform property. syntax transform-style: flat | preserve-3d | initial | inherit; flat : specifies that child elements will not preserve its 3D position.
